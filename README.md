# Site Oficial do Ofidismo Educacional

Aplicação institucional desenvolvida com React, Vite, TypeScript, Tailwind CSS, React Router DOM, Framer Motion e Lucide React.

## Rodando localmente

```bash
npm install
npm run dev
```

## Estrutura

- `src/pages`: páginas Home, Sobre, Animais e Aprendizado
- `src/components`: componentes reutilizáveis
- `src/data`: dados institucionais e catálogo educativo
- `src/layouts`: layout principal
- `src/routes`: rotas da aplicação
- `src/styles`: estilos globais

## Configuração de contato

Atualize os links de WhatsApp e Instagram em `src/data/site.ts`.
